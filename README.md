# WISH

**WISH — Where I Stay Human**

https://wish-brown.vercel.app/

A full-stack social platform built with React, Express.js, and PostgreSQL.

WISH is a minimal retro-themed platform where users can share thoughts and images, interact with content, and express themselves through a simple, distraction-free interface.

---

## Features

* User Authentication

  * Register new accounts
  * Secure login system

* Posts

  * Create posts with images
  * Add descriptions/captions
  * View posts in a responsive feed

* Feed

  * Dynamic feed displaying user-generated content
  * Individual post viewing through modal interface

* UI

  * Retro-inspired aesthetic
  * Glassmorphism effects
  * Responsive design
  * Custom pixel-style typography

## Installation

### Clone Repository

```bash
git clone https://github.com/sudosu321/WISH.git
cd WISH
```

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm start
```

---

## Environment Variables

Create a `.env` file in the server directory.

```env
DATABASE_URL=your_postgresql_connection_string
JWT_SECRET=your_secret_key
```
